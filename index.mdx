<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-lg dark:text-gray-400 text-gray-600 mb-8">Get Termy running in minutes</p>
  
  <Steps>
    <Step title="Install Termy">
      Download Termy for your platform:
      
      <CodeGroup>
      ```bash macOS (Homebrew)
      brew install --cask termy
      ```
      
      ```bash Linux
      curl -fsSL https://termy.run/install.sh | sh
      ```
      
      ```bash Windows
      # Download from https://termy.run/#download
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Launch and configure">
      Open Termy and customize your settings:
      
      ```bash
      # Open settings with Cmd+, (macOS) or Ctrl+, (Linux/Windows)
      # Or use the CLI
      termy-cli -edit-config
      ```
      
      <Tip>Configuration is stored in `~/.config/termy/config.txt`</Tip>
    </Step>
    
    <Step title="Start using powerful features">
      Try out key features:
      
      - **Command Palette**: Press `Cmd+P` (macOS) or `Ctrl+P` to access all commands
      - **Tabs**: Create new tabs with `Cmd+T` or `Ctrl+T`
      - **Search**: Find text in output with `Cmd+F` or `Ctrl+F`
      - **Themes**: Switch themes with the command palette or `termy-cli -list-themes`
      
      <Card title="Explore Configuration" icon="gear" href="/configuration/overview">
        Learn about all available configuration options
      </Card>
    </Step>
  </Steps>
</div>
